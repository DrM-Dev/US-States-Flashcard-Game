# States Flash Card Game 0.1 
*A game to help you memories the 50 states of the united states of America!*

_________________
This is a flash-cards game with accurate images for USA states on the map from (wikipedia)
with a grading system based on how many states you know from 50!
:)

_________________
to build/modify this code you need the following packages:
1. Pandas
2. Tkinter
3. Customtkinter
